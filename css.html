c3选择器的分类

基本选择器

通配符 *{}

元素(标签)选择器 div{}等等

id选择器 #id{}

类选择器 .class{}

组合选择器 div, .class, #id{}

层次选择器

后代选择器 div p{}

子选择器 div>p{}

相邻兄弟选择器(选择后一个div) .class+div{}

通用兄弟选择器(选择后面所有div) .class~div{}

伪元素选择器

Element:link{} 选择定义了超链接但链接还未被访问过的元素

Element:visited{} 选择定义了超链接并且链接已经被访问过的元素

Element:active{} 选择匹配的Element元素，且元素被激活，常用在锚点和按钮上

Element:hover{} 选择鼠标停留的匹配的Element元素

Element:focus{} 选择匹配的Element元素，且元素获得焦点

目标伪类选择器 Element:target 配合a标签的锚点链接使用

语言伪类选择器 Element:lang(language) 配合<html lang="language"></html>

状态伪类选择器 Element:checked,Element:enabled,Element:disabled 配合input标签使用

结构伪劣选择器

Element:first-child,Element:last-child,Element:root,Element:nth-child(n),Element:nth-last-child(n),Element:nth-of-type(n),Element:nth-last-of-type(n),Element:first-of-type,Element:last-of-type,Element:only-child,Element:only-of-type,Element:empty

Element:first-child 选择第一个子元素

Element:last-child 选择最后一个子元素

Element:nth-child(n) 选择第n个子元素 从1开始

其中的n可以是数值(从1开始)，也可以是包含n的表达式，也可以是odd(奇数)，even(偶数)

Element:nth-last-child(n) 选择倒序第n个子元素

Element:nth-of-type(n) 选择Element父元素中第n个Element元素

Element:nth-last-of-type(n) 同上 倒序方式

Element:first-of-type 选择Element父元素中第1个Element元素

Element:last-of-type 选择Element父元素中最后一个Element元素

Element:only-child 选择Element父元素中只有唯一一个Element的元素

Element:only-of-type 选择Element父元素中只有一个Element的元素，Element父元素可以有其他元素

Element:empty 选择没有内容的元素（包括空格）

Element:root 选择html

伪类选择器

在css3中，为了区分伪元素于伪类选择器，采用了::的写法，但:的写法在目前主流浏览器中也支持

注意： IE6-8只支持:的写法

::first-letter 选择文本块的第一个字

::first-line 选择文本块的第一行文字

::before 在标签的最前面插入内容

::after 在标签的最后面插入内容 (清浮动常用)

万能清除浮动


					.clearfix:after {
						content: ".";
						display: block;
						height: 0;
						clear: both;
						visibility: hidden;
					}

属性选择器

Element[attr] 选择Element中包含attr属性的元素

Element[attr=val] 选择Element中包含attr属性并且值为val的元素

Element[attr|=val] 选择Element中包含attr属性并且值为val或以val-开头的元素

Element[attr~=val] 选择Element中包含attr多个属性中包括val的元素

Element[attr*=val] 选择Element中包含attr属性中包含val的元素

Element[attr^=val] 选择Element中包含attr属性并且值以val开头的元素

Element[attr$=val] 选择Element中包含attr属性并且值以val结尾的元素

最后，关于移动端布局中的rem布局

rem布局

指相对于根元素（即html元素）的字体大小的单位进行布局。

是CSS3新增的一个相对单位，使用rem为元素设定字体大小时，相对大小对比的是HTML根元素。 用处：针对一种手机屏幕布局做到适应常见的手机屏幕。

支持。

目前，除了

IE8

及更早版本外，所有浏览器均已支持rem。

使用方式

利用css3 尺寸单位 VW

1VW = 1%的屏幕宽度

相对于iPhone6的设计稿 375px的逻辑像素

计算方式（以1rem=100px计算）： 100px/font-size VW = 375px/100VW

设置html font-size: 0.2667VW

40px 的宽度就可以设置成 0.4rem 了
